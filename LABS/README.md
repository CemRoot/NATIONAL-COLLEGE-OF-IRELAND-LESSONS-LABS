## ğŸš€ Getting Started with Labs

### Prerequisites

Ensure you have the following installed:
```bash
python >= 3.8
jupyter notebook
pandas
numpy
matplotlib
seaborn
scikit-learn
```

### Running a Lab

1. **Navigate to the specific lab directory**:
   ```bash
   cd "Programming for Artificial Intelligence (MSCAI1) LAB"/LAB1
   ```

2. **For Python scripts**:
   ```bash
   python main.py
   ```

3. **For Jupyter notebooks**:
   ```bash
   jupyter notebook
   # Open the desired .ipynb file
   ```

4. **Install lab-specific dependencies** (if requirements.txt exists):
   ```bash
   pip install -r requirements.txt
   ```
